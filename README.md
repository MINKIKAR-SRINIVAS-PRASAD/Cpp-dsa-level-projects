# 100 C++ DSA Projects for Resume

This collection contains 100 Data Structures and Algorithms implementations in C++.

## Categories:

### Arrays (01-15)
- Array Rotation, Kadane's Algorithm, Two Sum, Dutch National Flag
- Merge Intervals, Stock Buy Sell, Trapping Rainwater
- Product Except Self, Next Permutation, Spiral Matrix, etc.

### Linked Lists (16-25)
- Reverse List, Detect Cycle, Merge Sorted Lists
- Middle Element, Remove Nth Node, Palindrome Check
- Add Two Numbers, Intersection, Rotate, Flatten

### Stacks & Queues (26-35)
- Stack/Queue Implementations
- Valid Parentheses, Next Greater Element
- Min Stack, Sliding Window Maximum, Postfix Evaluation

### Trees (36-50)
- Traversals (Inorder, Preorder, Postorder, Level Order)
- Height, Diameter, Balanced Tree
- LCA, Binary Tree Paths, Symmetric Tree
- Max Path Sum, Zigzag Level Order, Serialize/Deserialize

### Graphs (51-65)
- BFS, DFS, Cycle Detection
- Topological Sort, Dijkstra, Bellman-Ford
- MST (Prim's, Kruskal's), Floyd-Warshall
- Bipartite, Bridges, Articulation Points, SCC, Graph Coloring

### Dynamic Programming (66-80)
- Fibonacci, Knapsack, LCS, Edit Distance
- Coin Change, LIS, Matrix Chain Multiplication
- Rod Cutting, Egg Dropping, Palindrome Partitioning
- Word Break, Subset Sum, Min Path Sum, Unique Paths

### Searching & Sorting (81-90)
- Binary Search, Merge Sort, Quick Sort, Heap Sort
- Counting Sort, Radix Sort
- Search in Rotated Array, Peak Element
- 2D Matrix Search, Kth Largest Element

### Miscellaneous (91-100)
- N-Queens, Sudoku Solver, Rat in Maze
- Trie, LRU Cache, Segment Tree, Fenwick Tree
- Union Find, Manacher's Algorithm, KMP Algorithm

## Compilation:
```bash
g++ -std=c++17 <filename>.cpp -o output
./output
```
